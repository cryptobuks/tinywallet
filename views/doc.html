Tiny Wallet

the easist use and extensable crypto wallet

User only need to remember the id and password to login and transfer.

PrivateKey was encrypted and save to database. User need to input password to sign transaction everytime.

You may think, It was easy and danger to loss password.

Yes and no, password would loss easy. but we could setup different limit level passwords. and we could know transaction was signed by which password
