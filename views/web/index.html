TinyWallet is a Self-Hosted programmable crypto wallet
Self-Hosted Usage
- Transation Approve, role-based management, separate (view|submit|approve) roles
- Assert Audit, erc20 tokens, nfts transactions audit by tags, users, logs
- Coin Currency Exchange, exchange between coin and currency
- Nft Shop, customized nft shop
Api-First Design
1 create or import wallet /api/wallet
2 create api account /api/wallet/:address/account
3 submit transation /api/wallet/:address/transation
4 event webhook http://abc.xyz/webhook
